# 📝 Notas Pro - Extensión de Chrome

¡Mi primera extensión! Esta herramienta permite guardar notas personalizadas por cada sitio web que visitas. 
Las notas se quedan guardadas localmente en tu navegador.

## ✨ Características
- Notas independientes por URL.
- Guardado persistente (usa Chrome Storage).
- Interfaz sencilla y ligera.

## 🚀 Cómo instalarla
1. Descarga este repositorio.
2. Ve a `chrome://extensions/`.
3. Activa el **Modo Desarrollador**.
4. Haz clic en **Cargar descomprimida** y selecciona la carpeta.
